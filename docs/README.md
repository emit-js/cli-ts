
@emit-js/cli
============

emit cli

![cli](media/cli.gif)

Install
-------

```bash
npm install @emit-js/cli
```

Syntax
------

```bash
emit EVENT [STRING_ARGS...] [--OPTIONS]
```

Example
-------

Let's run the [`@emit-js/spawn`](https://github.com/emit-js/spawn) event from the CLI:

```bash
npm install -g @emit-js/cli @emit-js/spawn
emit @emit-js/spawn --command=ls
```
